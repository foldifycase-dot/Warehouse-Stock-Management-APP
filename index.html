How to Bulk-Tag in Shopify
1
Go to Shopify Admin → Products. Filter to your US warehouse products and select all with the checkbox.
2
Click Actions → Add tags. Type warehouse:us and save. Repeat for UK with warehouse:uk and AU with warehouse:au.
3
For dropship-only products add tag dropship. These are automatically hidden from every warehouse view.
Adding a New Market Later
Tag products with warehouse:xx in Shopify, then update this app to add a new tab. No other code changes needed.
